# ESP32 UART actuator firmware addon
This is the ESP32 side expected by the Pi bridge.

## Important
This addon needs the old project files:
- `esp32_ros_pcb/hardware_config.h`
- `esp32_ros_pcb/hardware_fns.cpp`
- `esp32_ros_pcb/motor_ctrl/`
- `esp32_ros_pcb/rc.h`

**This folder is not fully standalone by itself**
